# IA2-Tarea6
Multiclass classification using a multilayer neural network with Optuna-based hyperparameter optimisation on the Glass Identification dataset

Implementación de una red neuronal profunda para clasificar tipos de vidrio basándose en su composición química, aplicando optimización de hiperparámetros y early stopping.

## Archivos
* IA2_Tarea6.ipynb: Libreta con el preprocesamiento, entrenamiento y análisis.
* glass.data / glass.names: Dataset de identificación de vidrios.
* modelo_vidrio_mlp.keras: Modelo final entrenado.
